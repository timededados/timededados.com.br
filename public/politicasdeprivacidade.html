Aviso de Privacidade – PR QUEST

O que fazemos com as suas informações

Fizemos o máximo para explicar de forma clara e simples quais dados pessoais precisaremos de você e o que vamos fazer com cada um deles. Abaixo estão os principais pontos, mas estamos sempre à disposição para tirar qualquer dúvida que você tenha pelo e-mail contato@timededados.com.br. Para falar especificamente sobre seus dados pessoais, possuímos um canal específico: privacidade@timededados.com.br.

Quem é o responsável pelo tratamento de dados?

A contratante intermediária é a controladora dos dados pessoais fornecidos pelos alunos, ou seja, é quem decide quais dados serão coletados e como serão utilizados na plataforma PR QUEST. A Time de Dados Ltda atua como operadora do tratamento, sendo responsável pelo processamento seguro e adequado dessas informações, conforme orientação da controladora. Também poderão ser envolvidos operadores terceiros, como provedores de infraestrutura tecnológica (como o Google Cloud Platform).

Como protegemos seus dados?

A Time de Dados implementa as boas práticas recomendadas pela Autoridade Nacional de Proteção de Dados (ANPD), utilizando tecnologias e procedimentos que garantem a segurança, confidencialidade e integridade das informações. Os dados são armazenados em servidores do Google Cloud Platform, localizados fora do Brasil. Esses servidores possuem alto padrão de segurança e são acessados de forma controlada.

Quais dados são coletados?

Para uso do PR QUEST, serão coletadas informações básicas necessárias para o funcionamento da plataforma:

  • Nome completo
  • Número de telefone com DDD
  • Tipo de pacote contratado (Intensivo, Quest, Masterclass)

Esses dados são fornecidos pela contratante intermediária para cadastro inicial dos usuários.

Para que usamos seus dados?

  • Permitir o acesso à plataforma via WhatsApp
  • Identificar o seu perfil e liberar o conteúdo correspondente ao pacote adquirido
  • Gerar relatórios de desempenho e uso
  • Realizar manutenção, correção de erros e melhorias no sistema

Com quem compartilhamos seus dados?

Não compartilhamos seus dados com terceiros, exceto:

  • Operadores necessários para execução do serviço (como o GCP);
  • Por obrigação legal ou judicial;
  • Em casos expressamente autorizados pela contratante intermediária.

E quanto ao WhatsApp?

O WhatsApp é uma plataforma independente e externa à Time de Dados e à contratante intermediária. Por isso, recomendamos que você leia os termos de uso e a política de privacidade do WhatsApp antes de utilizar o PR QUEST.

Seus registros de acesso serão armazenados?

Sim. Conforme obrigação legal (Lei 12.965/2014 e LGPD), os registros de acesso serão armazenados por pelo menos 6 meses em ambiente seguro e controlado.

Quais são seus direitos?

Você tem o direito de:

  • Solicitar confirmação do tratamento de dados
  • Acessar, corrigir ou atualizar seus dados
  • Solicitar anonimização, bloqueio ou exclusão de dados
  • Revogar consentimentos e solicitar a portabilidade dos dados
  • Solicitar informações sobre com quem os dados foram compartilhados

Esses pedidos podem ser feitos diretamente à contratante intermediária, controladora dos dados, ou encaminhados para a Time de Dados, que os direcionará adequadamente.

Transferência Internacional de Dados

Os dados dos usuários poderão ser armazenados em servidores localizados fora do Brasil (EUA), conforme previsto na LGPD, desde que garantido o mesmo nível de proteção exigido pela legislação brasileira.

Atualizações deste Aviso de Privacidade

Este Aviso pode ser atualizado a qualquer momento. Sempre que houver alteração relevante, os usuários serão comunicados.
